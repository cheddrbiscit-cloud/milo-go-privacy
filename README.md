# Milo Go - Privacy Policy

**Last Updated: March 30, 2026**

## Overview
Milo Go is a mileage tracking app that stores all data locally on your device.

## Data Collection
Milo Go collects the following data, which remains on your device:
- **Location Data**: Used to track trip routes and distances while driving. Location is accessed only when you start a trip or enable automatic tracking.
- **Photos** (optional): Used only if you attach odometer photos to readings.

## Data Storage
All trip data, routes, and settings are stored locally on your device using Apple's SwiftData framework. No data is transmitted to external servers.

## Third-Party Services
Milo Go does not use any third-party analytics, advertising, or tracking services.

## On-Device AI
Milo Go uses Apple's on-device Foundation Models to suggest trip classifications. This processing happens entirely on your device.
